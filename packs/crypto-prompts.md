# Crypto Trading AI Prompt Kit v3.0

## Market Analysis
```
Analyze the current Bitcoin market structure. Consider:
- Price action relative to 20-day and 50-day moving averages
- RSI divergence on the 4h and daily timeframes
- Funding rates across major exchanges
- Open interest changes
- Whale wallet movements (top 100 addresses)
Provide a structured assessment: bullish, bearish, or neutral with confidence levels.
```

## Risk Management
```
Given a portfolio with $1000 total value, currently holding:
- 0.015 BTC (45% of portfolio)
- 0.5 ETH (30% of portfolio)
- SOL (25% of portfolio)

Calculate optimal position sizing for a new entry, considering:
- Maximum 2% portfolio risk per trade
- Current volatility (ATR-based)
- Correlation between assets
- Recommended stop-loss and take-profit levels
```

## Technical Analysis
```
Generate a technical analysis report for SOL/USDT including:
1. Trend analysis (short, medium, long term)
2. Key support and resistance levels
3. Volume profile analysis
4. Momentum indicators (RSI, MACD, Stochastic)
5. Chart pattern identification
6. Probability-weighted price targets
Format as a professional trading desk report.
```

## Sentiment Analysis
```
Analyze crypto market sentiment based on:
- Twitter/X trending topics and sentiment
- Fear & Greed Index components
- Reddit r/CryptoCurrency top posts
- Bitcoin dominance trend
- Stablecoin supply ratio (SSR)
Provide a sentiment score and trading implications.
```

## Alpha Generation
```
Identify 3 undervalued crypto projects based on:
- Developer activity (GitHub commits)
- On-chain metrics (active addresses, TVL)
- Tokenomics (vesting schedule, inflation)
- Partnership announcements
- Social media momentum
Risk-assess each pick on a 1-5 scale.
```
